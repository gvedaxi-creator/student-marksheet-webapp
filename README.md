Project Overview

The Smart Campus Lost & Found System is a web-based application developed using Laravel to provide a centralized and organized platform for managing lost and found items within a college campus.

The system enables students to report lost or found items with images and automatically suggests potential matches using category-based and keyword-based matching logic.

Objective

To design and implement a structured digital solution that simplifies the process of reporting, tracking, and recovering lost items through intelligent match suggestions and efficient database management.

Key Features

User authentication system

Lost and found item posting

Image upload functionality

Automatic match suggestion

Match confirmation and status tracking

Admin monitoring panel

Technology Stack

Backend: Laravel (PHP)

Frontend: Blade, Bootstrap

Database: MySQL

Development Tools: XAMPP, VS Code

Conclusion

The system enhances efficiency in managing lost and found items by integrating structured data management with intelligent matching logic, providing a practical and scalable solution for campus environments.
